# Desynced 命令台模组 (Command Console Mod)

## 简介

这是一个为 Desynced 游戏开发的命令行模组，允许玩家通过游戏内置的聊天系统使用文本命令来修改地图参数和游戏设置。

## 功能

- **地图设置查看和修改**: 查看所有可用的地图设置，或修改特定的设置
- **游戏速度控制**: 查看和修改游戏模拟的速度
- **游戏信息显示**: 查看当前的游戏时间、日期、年份等信息
- **地图种子查询**: 查看或修改地图生成种子
- **帮助系统**: 内置的命令帮助系统

## 安装

1. 将此模组放在 Desynced 的 mods 文件夹中
2. 在游戏中启用此模组
3. 开始新的游戏或在现有游戏中使用

## 使用方法

所有命令都以 `/` 前缀开头，在聊天栏中输入。

### 基础命令

#### `/help` - 显示帮助
显示所有可用的命令列表。
```
/help                  # 显示所有命令
/help <命令名>         # 显示特定命令的帮助
```

#### `/settings` - 查看地图设置
查看当前的地图设置。
```
/settings              # 显示所有设置
/settings <设置名>     # 显示特定设置
```

**常用地图设置:**
- `seed` - 地图生成种子
- `blight_threshold` - 污染阈值
- `pollution_limit` - 污染限制
- 其他根据场景的具体设置

#### `/set` - 修改地图设置
修改地图的具体设置。
```
/set <设置名> <值>
```

**示例:**
```
/set seed 12345                    # 设置地图种子
/set blight_threshold 0.5          # 设置污染阈值
/set pollution_limit 100           # 设置污染限制
```

#### `/speed` - 游戏速度
查看或修改游戏速度（模拟倍速）。
```
/speed                 # 查看当前游戏速度
/speed <倍速值>        # 设置游戏速度
```

**示例:**
```
/speed                 # 显示当前速度
/speed 1              # 正常速度
/speed 2              # 2倍速
/speed 0.5            # 0.5倍速（慢速）
```

#### `/info` - 游戏信息
显示当前游戏的相关信息。
```
/info
```

显示的信息包括:
- 当前游戏的 Tick 数
- 已过天数
- 当前年份
- 日出和日落时间

#### `/seed` - 地图种子
查看或修改地图的生成种子。
```
/seed              # 查看当前种子
/seed <新种子>    # 设置新种子
```

**注意:** 修改种子后可能需要重新加载地图才能生效。

#### `/version` - 版本信息
显示模组的版本信息。
```
/version
```

## 使用示例

### 示例 1: 基本使用流程
```
/help                        # 查看可用命令
/info                        # 检查当前游戏进度
/settings                    # 查看所有可用设置
/settings seed              # 查看地图种子
```

### 示例 2: 修改游戏速度进行快速建设
```
/speed 2                     # 设置 2 倍速
# 进行快速建设...
/speed 1                     # 恢复正常速度
```

### 示例 3: 调整地图参数
```
/settings blight_threshold   # 查看污染阈值
/set blight_threshold 0.7    # 提高污染阈值
/info                        # 验证设置
```

## 技术细节

### 架构
- **模块化设计**: 命令通过统一的处理系统管理
- **类型检测**: 自动识别设置的数据类型（布尔值、数值、字符串）
- **错误处理**: 完善的错误提示和异常捕获

### 支持的设置类型
- **布尔值**: 接受 true/false、yes/no、1/0、on/off
- **数值**: 整数和浮点数
- **字符串**: 直接传递

### 消息系统
模组使用 Desynced 的聊天系统来：
1. 拦截以 `/` 开头的消息
2. 解析命令和参数
3. 执行相应的操作
4. 返回执行结果

## 故障排除

### 命令无效
- 检查命令前缀是否正确（应为 `/`）
- 使用 `/help` 查看确切的命令名称和格式
- 确保模组已正确加载

### 设置无法修改
- 确认设置名称正确（使用 `/settings` 查看）
- 检查值的类型是否正确
- 某些设置可能在游戏运行时无法修改

### 游戏速度修改无效
- 设置的速度值应为非负数
- 检查当前游戏是否允许修改速度
