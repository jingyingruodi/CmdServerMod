## 🚀 快速开始（3步）

### 1️⃣ 安装
- 确保模组在 Desynced mods 文件夹中
- 启动游戏并启用模组

### 2️⃣ 打开游戏
- 创建新游戏或打开现有游戏
- 游戏加载完毕后模组自动激活

### 3️⃣ 使用命令
在聊天栏输入：
```
/help          # 查看所有可用命令
/help set      # 查看特定命令的帮助
/settings      # 查看当前地图设置
```

## 📖 常用命令速查

### 查看信息
```
/help              # 显示所有命令帮助
/info              # 显示游戏当前信息（时间、日期等）
/settings          # 显示所有地图设置

/speed             # 显示当前游戏速度
/seed              # 显示地图种子
/version           # 显示模组版本
```

### 修改设置
```
/set seed 12345          # 设置地图种子为 12345
/set blight_threshold 0.5 # 设置污染阈值为 0.5
/speed 2                 # 设置游戏速度为 2 倍
/seed 54321             # 改变地图种子为 54321
```

### 获取帮助
```
/help                # 显示所有命令列表
/help set           # 查看 /set 命令的详细说明
/help settings      # 查看 /settings 命令的详细说明
/help <命令名>      # 查看任何命令的帮助
```

## 🛠️ 扩展和定制

### 想添加新命令？
参考 `EXTENSION_GUIDE.lua` 中的示例：

```lua
register_command("mycommand", "命令描述", "[参数]", 
    function(args, sender_id)
        print_to_chat("[CMD] 命令执行结果")
    end
)
```

### 支持的高级特性
- 数据持久化（使用 Map.GetSave()）
- UI 交互（使用 UI.Run()）
- 延迟执行（使用 Map.Defer()）
- 事件处理（使用 MapMsg）

## ❓ 常见问题解答

### Q: 如何查看所有可用的命令？
A: 在聊天中输入 `/help`

### Q: 如何修改地图设置？
A: 
1. 首先使用 `/settings` 查看所有可用设置
2. 使用 `/set <设置名> <值>` 进行修改
3. 例如: `/set seed 12345`

### Q: 如何加快游戏速度？
A: 输入 `/speed 2` （2倍速）或其他倍数值

### Q: 修改的设置会被保存吗？
A: 设置在游戏存档中被保存，重新加载游戏后仍然有效

### Q: 可以添加新的命令吗？
A: 可以！查看 `EXTENSION_GUIDE.lua` 了解如何添加自定义命令
