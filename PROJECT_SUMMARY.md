3. **`/set`** - 地图设置修改
   - 支持布尔值、数值、字符串类型

   - 完整的错误处理

4. **`/speed`** - 游戏速度控制

   - 设置新的游戏速度（支持浮点数）

5. **`/info`** - 游戏信息显示
   - 显示当前游戏 Tick
   - 显示已过天数
   - 显示当前年份
   - 显示日出日落时间

6. **`/seed`** - 地图种子管理
   - 查看当前地图种子
   - 修改地图种子

7. **`/version`** - 版本信息
   - 显示模组版本和描述

## 技术架构

### 核心设计模式
1. **注册-执行模式**：命令通过 `register_command()` 注册，通过 `execute_command()` 执行
2. **表驱动设计**：命令存储在 `commands` 表中，便于管理和扩展
3. **函数式编程**：大量使用高阶函数和闭包

### 主要函数
- `split_string()` - 字符串分割
- `print_to_chat()` - 聊天消息输出
- `execute_command()` - 命令执行
- `register_command()` - 命令注册

### API 使用
- **Map 模块**: 获取和修改游戏状态
- **UI 模块**: 与聊天系统交互
- **Tool 模块**: 数据处理和转换
