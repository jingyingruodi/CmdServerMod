# 快速开始指南 - Desynced 命令台模组

## 安装步骤

1. 确认模组文件在 Desynced mods 文件夹中
2. 启动 Desynced 游戏
3. 创建新游戏或打开现有游戏
4. 在模组列表中启用 "Command Console"
5. 开始游戏

## 第一次使用

### 步骤 1: 查看可用命令
在游戏聊天中输入：
```
/help
```

### 步骤 2: 查看当前设置
```
/settings
```

### 步骤 3: 查看游戏信息
```
/info
```

### 步骤 4: 修改游戏速度
```
/speed 2
```

## 常见操作

### 加快游戏速度以进行快速建设
```
/speed 2      # 设置为 2 倍速
# ... 进行建设 ...
/speed 1      # 恢复正常速度
```

### 查看地图的具体设置
```
/settings seed                    # 查看地图种子
/settings blight_threshold        # 查看污染阈值
```

### 修改地图参数
```
/set blight_threshold 0.5         # 降低污染阈值
/set pollution_limit 100          # 设置污染限制
```

### 获取帮助
```
/help                  # 显示所有命令
/help settings         # 显示 /settings 命令的帮助
/help set              # 显示 /set 命令的帮助
```

## 故障排除

### 问题：命令无法识别
**解决方案：**
- 确认以 `/` 开头
- 检查命令名称是否正确
- 使用 `/help` 查看有效的命令

### 问题：设置无法修改
**解决方案：**
- 检查设置名称是否存在（使用 `/settings`）
- 确认值的类型正确
- 某些设置可能需要重新加载游戏才能生效

### 问题：游戏速度修改无效
**解决方案：**
- 确认速度值为非负数
- 尝试设置为 1（正常速度）
- 检查游戏是否仍在运行

## 常用命令速查

| 命令 | 用途 |
|------|------|
| `/help` | 查看所有命令 |
| `/settings` | 查看所有设置 |
| `/set key value` | 修改设置 |
| `/speed [值]` | 修改游戏速度 |
| `/info` | 查看游戏信息 |
| `/seed [值]` | 查看/修改地图种子 |

## 提示和技巧

1. **保存常用配置**
   - 记下经常使用的设置值
   - 在游戏开始前预先设置

2. **使用速度控制进行时间管理**
   - 低速（0.5倍）进行精细操作
   - 高速（2-3倍）快速完成任务

3. **监控游戏进度**
   - 定期使用 `/info` 检查游戏时间
   - 观察日夜周期的变化

4. **实验性设置**
   - 在新游戏中测试不同的参数
   - 使用小地图测试修改效果

## 获取帮助

如需特定命令的详细帮助：
```
/help <命令名>
```

例如：
```
/help set              # 查看 /set 命令的详细说明
/help settings         # 查看 /settings 命令的详细说明
```
